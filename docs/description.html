# 🪙 RGX | Runescape Level-Up! ![RSLU Logo](https://github.com/DonnieDice/RunescapeLevelUp/blob/main/media/logo.png?raw=true) ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Runescape Level-Up!** brings an Old School RuneScape-inspired level-up jingle to World of Warcraft, replacing the default ding with a clean nostalgic RPG-style level-up hit. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) **The Kiwi Says:** "XP drop complete. Bwwiiiee." *** ## ✨ Features 🎵 RuneScape Sound - An Old School RuneScape-inspired level-up sound plays every time you gain a level. 🎚️ Quality Options - Switch between High, Medium, and Low quality variants. 🔇 Smart Muting - Auto-mutes WoW's default level-up sound so you only hear RSLU. 💬 Login Message - A branded RSLU login message appears on login and can be toggled on or off. 💾 Persistent Settings - Settings are saved between sessions. 💬 Slash Commands - Simple /rslu commands for testing and switching sound quality. 🎮 Multi-Version - Includes support for Retail, Wrath, Burning Crusade, and Classic Era. *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/rslu help # Show all available commands
/rslu test # Play test sound
/rslu enable # Enable the addon
/rslu disable # Disable the addon
/rslu welcome # Toggle login message
/rslu high # Use high quality sound
/rslu med # Use medium quality sound (default)
/rslu low # Use low quality sound
